# .github — DevSwap organization defaults

This repository holds the **organization-wide community health files and public profile** for the
[**DevSwap**](https://devswap.pro) organization. GitHub applies these defaults to every DevSwap
repository that does not define its own.

## Contents
- [`profile/README.md`](profile/README.md) — the public landing page shown at [github.com/DevSwap-org](https://github.com/DevSwap-org).
- [`SECURITY.md`](SECURITY.md) — responsible-disclosure policy (report privately to **security@devswap.pro**; private advisories enabled).
- [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) · [`CONTRIBUTING.md`](CONTRIBUTING.md) — community standards and contribution workflow.
- [`.github/CODEOWNERS`](.github/CODEOWNERS) · [`.github/PULL_REQUEST_TEMPLATE.md`](.github/PULL_REQUEST_TEMPLATE.md) · [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE) — review routing and templates.
- [`.github/FUNDING.yml`](.github/FUNDING.yml) — sponsorship configuration.

## About DevSwap
A **non-custodial protocol** for on-chain software-services settlement on **BNB Smart Chain** — settled
in USDT, with an immutable **97 % developer floor**, and a bilingual (EN / AR, RTL) dApp.

- 🌐 App: **https://devswap.pro**
- 📜 Contracts (public): **https://github.com/DevSwap-org/devswap-contracts**
- 📘 Docs: **https://github.com/DevSwap-org/devswap-docs**

> Report security issues **privately** — see [SECURITY.md](SECURITY.md). Never open a public issue for a vulnerability.
