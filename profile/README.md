# DevSwap

### On‑chain freelance marketplace — funds settled by smart contract, not a middleman

[**devswap.pro**](https://devswap.pro) · BNB Smart Chain · settled in USDT · bilingual (EN / AR, RTL)

</div>

---

DevSwap is a decentralized peer‑to‑peer marketplace for software services. A client locks the
budget **on‑chain in USDT**; the **smart contract** releases it to the developer only when the
client approves the delivery. No platform custody, no middleman holding funds.

### How it works
1. **Lock** — the client funds a task; USDT is locked in the smart contract.
2. **Deliver** — the developer accepts and submits the work.
3. **Release** — the client approves → the contract releases funds automatically; or opens a transparent dispute.

### Economics
- **97%** to the developer · **1.5%** platform fee · **1.5%** community **buyback‑and‑burn** of `$DSWP`.
- Total fee **3%** — settlement is enforced by the publicly deployed contract.

### Tech
Solidity 0.8.24 (Foundry) · Next.js 14 on **Cloudflare Workers** (OpenNext) · wagmi/viem · The Graph · IPFS.
